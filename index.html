Find the Largest Element using Divide and Conquer

Approach

1. Divide array into halves
2. Recursively find largest in each half
3. Compare and return the maximum

Pseudocode

Function FindMaxIndex(arr, low, high):
    if low == high:
        return low
    mid = (low + high) // 2
    left = FindMaxIndex(arr, low, mid)
    right = FindMaxIndex(arr, mid+1, high)
    return max(left, right)
